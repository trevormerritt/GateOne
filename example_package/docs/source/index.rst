The Example Packaged Gate One Plugin
====================================

.. moduleauthor:: Dan McDougall <daniel.mcdougall@liftoffsoftware.com>

This documentation covers the example packaged plugin.  It's purpose is to
demonstrate how to create an installation package for Gate One that can include
plugins or applications.

Just make a copy of the ``example_package`` directory and start renaming things,
add your own code, etc etc.
